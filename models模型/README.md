# 机械模型与尺寸 Mechanical Models

**当前配置 / Current configuration:** 最终装配只需要摄像头视觉支架，不需要超声波支架或编码器信号线。The final assembly requires a camera mount for vision, with no ultrasonic mounts or encoder signal wiring.

基础底盘：RF-A101HE-109010203 阿克曼智能车。

已确认结构：前轮舵机转向、四根转向拉杆、四轮驱动、前后差速器、中间传动轴、带霍尔编码器接口的电机；当前仅使用电机驱动功能，编码器接口未接入。底部小亚克力板预留 Arduino UNO R3 安装孔，顶部用于安装Orange Pi、USB摄像头和电源模块。

最终参赛车辆增加主控、电池、USB摄像头和视觉支架后，应重新测量总长、总宽、总高和总质量。

当前目录包含 `HSP94182层板.dxf`，这是可编辑的二维 CAD/DXF 文件，可作为层板加工与孔位复核依据。文件名和几何内容应在最终装车后与实际使用的层板逐项核对；仓库暂不声称该文件已经代表最终参赛车辆版本。

正式提交前还应加入：

- 底盘俯视尺寸图（轴距、轮距、轮径、总长、总宽、总高）；
- 摄像头支架 CAD/STL、安装高度与俯仰角；
- 控制板、电池和稳压模块固定件；
- 舵机中位、拉杆孔位和最大安全转角；
- 最终装配图与物料表；
- 文件单位、打印方向、材料和紧固件规格。

不要用未经实测的商品页面参数替代参赛车辆的最终尺寸。
