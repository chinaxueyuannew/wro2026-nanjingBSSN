# 机械模型与尺寸 Mechanical Models

基础底盘：RF-A101HE-109010203 阿克曼智能车。

已确认结构：前轮舵机转向、四根转向拉杆、四轮驱动、前后差速器、中间传动轴、霍尔编码器电机；底部小亚克力板预留 Arduino UNO R3 安装孔，顶部前端有部分激光雷达孔位，后部通孔可安装其他主控板。

## 团队确认的底盘尺寸与性能

| 项目 | 数值 |
|---|---:|
| 整车外形 | 260 × 140 × 85 mm |
| 去防撞棉主体长度 | 246 mm |
| 轴距 | 174 mm |
| 轮距 | 123 mm |
| 车轮直径 | 47 mm |
| 离地间隙 | 6 mm |
| 整车基础重量 | 0.7 kg |
| 标称负载 | 0.3 kg |
| 最小转弯半径 | 475 mm |
| 标称爬坡角 | 小于 20° |

最终参赛车辆增加主控、电池、摄像头、超声波和支架后，应重新测量总长、总宽、总高和总质量。

当前目录包含 `HSP94182层板.dxf`，这是可编辑的二维 CAD/DXF 文件，可作为层板加工与孔位复核依据。文件名和几何内容应在最终装车后与实际使用的层板逐项核对；仓库暂不声称该文件已经代表最终参赛车辆版本。

正式提交前还应加入：

- 底盘俯视尺寸图（轴距、轮距、轮径、总长、总宽、总高）；
- 传感器支架 CAD/STL 与安装尺寸；
- 控制板、电池和稳压模块固定件；
- 舵机中位、拉杆孔位和最大安全转角；
- 最终装配图与物料表；
- 文件单位、打印方向、材料和紧固件规格。

不要用未经实测的商品页面参数替代参赛车辆的最终尺寸。
