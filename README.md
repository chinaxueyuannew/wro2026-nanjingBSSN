# 工程材料 Engineering Materials

本仓库包含参加 **2026年WRO未来工程师（Future Engineers）** 比赛的自动驾驶车辆模型的工程材料。

This repository contains engineering materials of a self-driven vehicle's model participating in the WRO Future Engineers competition in the season 2026.

## 目录结构 Content

* `t-photos团队照片` — 包含2张团队照片（1张正式照 + 1张全员趣味照）
* `v-photos车辆照片` — 包含6张车辆照片（各角度、顶部、底部）
* `video视频` — 包含 `video.md` 文件，内有驾驶演示视频的链接
* `schemes原理图` — 电机械组件的原理图（JPEG/PNG/PDF格式），展示所有电子元件、电机及其连接方式
* `src源代码` — 参赛所有编程组件的控制软件源代码
* `models模型` — 用于生产车辆零件的3D打印/激光切割/CNC文件（如无需可删除此目录）
* `other其他` — 其他辅助文件，如SBC/SBM连接文档、数据集、硬件规格、通信协议等（如无需可删除此目录）

## 技术说明 Introduction

_此部分由参赛队填写，包括代码的技术说明：代码由哪些模块组成、各模块与车辆电机械组件的关系、如何构建/编译/上传代码到车辆控制器。_
