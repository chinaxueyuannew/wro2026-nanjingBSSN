## 规则要求的团队照

- [x] `team-official.jpg`：全体参赛队员正面、清晰、背景整洁；
- [ ] `team-funny.jpg`：全体参赛队员入镜的趣味照片。

后续趣味团队照应确保所有参赛队员均清晰入镜，同时避免公开不必要的个人隐私。
