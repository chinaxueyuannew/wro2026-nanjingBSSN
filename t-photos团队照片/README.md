# 团队照片选择说明

WRO 官方模板要求团队照片目录包含一张正式团队照和一张全员趣味照。当前目录有多张候选照片，最终提交前应完成筛选并使用明确文件名：

- `team-official.jpg`：全体参赛队员正面、清晰、背景整洁；
- `team-funny.jpg`：全体参赛队员入镜的趣味照片。

筛选后建议把其他候选图移入 `candidates/`，不要删除原始照片。README 中可补充团队名称、队员分工和指导教师，但不要公开不必要的个人隐私。

## 建议分工记录

| 角色 | 主要职责 | 姓名 |
|---|---|---|
| 机械与制造 | 底盘、安装件、CAD、装配 | 待填写 |
| 电气与传感器 | 供电、接线、传感器标定 | 待填写 |
| 软件与算法 | 状态机、闭环控制、测试日志 | 待填写 |
| 项目与文档 | Git、工程日志、演示视频 | 待填写 |

角色可以重叠，但每名队员都应能解释整车的基本工作原理和本人负责部分。
