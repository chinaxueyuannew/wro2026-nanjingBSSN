# 标定与参数调优手册

Calibration and Parameter-Tuning Guide

**当前范围 / Current scope:** 当前只标定舵机、摄像头、视觉参数和开环速度命令映射。Current calibration covers the steering servo, camera, visual parameters and open-loop speed-command mapping; ultrasonic and encoder calibration is not used.

## 标定总原则

一次只改变一个参数；每次记录 Git 提交号、电池电压、场地、轮胎状态和测试结果。当前只使用视觉，应按“机械与舵机 → 摄像头 → 视觉参数 → 电机命令映射 → 完整回合”的顺序标定。

## 1. 舵机标定

1. 抬起车辆并断开电机动力。
2. 程序只输出 90°，调整舵臂和拉杆，使车轮接近直行。
3. 以 2° 为步长向左右移动，找到不发生机械顶死的极限。
4. 各向内保留约 3°–5°软件安全余量。
5. 低速直行 2 m，通过中位微调消除持续偏转。
6. 将最终值写入 `SERVO_RIGHT/CENTER/LEFT` 并拍照留档。

## 2. 编码器标定（当前不适用）

底盘具备霍尔编码器接口，但当前车辆未接线、未读取，也不做速度闭环。因此比赛配置不需要编码器标定；厂家CPR和减速比只保留在机械参考资料中。

## 3. 超声波标定（当前不适用）

当前不安装前方或右侧超声波，比赛配置不进行超声波测距或巡墙标定。仓库早期程序中的超声波参数不得复制到当前视觉方案。

## 4. 视觉速度命令映射

1. 把车辆架空，将视觉目标速度限制在较小范围，确认电机方向。
2. 在地面直线行驶固定距离，分别记录多个 `speed` 命令对应的平均速度和停车距离。
3. 使用满电、中等电量和允许的低电量重复测试，确认无编码器反馈时速度变化仍处于安全范围。
4. 设置比赛最大速度、转弯限速、障碍附近减速和视觉失效停车参数。
5. 不使用速度PI术语描述当前方案；实际控制是视觉目标量到PWM的开环映射。

## 5. 视觉转向调参

1. 固定低速，分别验证道路密度转向和红绿信标转向的正负方向。
2. 从较小最大转向与权重开始，逐步增加到能够回到目标轨迹。
3. 若出现持续蛇形，降低道路转向权重、最大转向或提高时间平滑。
4. 测试直道、弯道、画面边缘、阴影、反光、红绿障碍同时出现和短时目标丢失。
5. 最终参数以连续完整回合成功率和零碰撞为依据，而不是单圈最快时间。

## 6. 视觉弯道与恢复参数

转弯时机由道路区域、边界位置和CW/CCW策略判断，不使用 `FRONT_TURN_CM`。先用低速验证弯道目标轨迹，再调整道路区域阈值、最大转向、减速区和恢复状态持续时间。左右转向机械可能不对称，应分别保存顺时针和逆时针参数，并验证摄像头遮挡或目标丢失后能够停车而不是盲目直行。

## 7. 摄像头标定

1. 在最终安装位置固定摄像头，锁紧焦距和支架后再标定。
2. 使用棋盘格从画面中央、四角、不同距离采集不少于 15–20 张清晰图像。
3. 计算内参矩阵和畸变系数，保存标定文件并记录图像分辨率。
4. 检查去畸变后的直线是否接近直线；不要过度裁切导致障碍物提前离开视野。
5. 在比赛场地分别采集红柱、绿柱、黑墙、浅色地垫和阴影样本。
6. 优先在 HSV 或 Lab 空间设定颜色阈值，记录不同现场照明下的范围。
7. 用独立测试集计算误检率、漏检率和检测延迟；阈值调整不能只看一张图。
8. 核对自动曝光和白平衡是否可锁定。若无法锁定，应扩大光照测试条件。

160°广角镜头在画面边缘畸变明显，因此障碍物的像素位置不能直接线性等同于真实角度。用于转向控制前，应去畸变或通过实测像素到视角查找表建立映射。
